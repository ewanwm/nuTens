Detailed Description

Basic tensor class.

Tensor defines a basic interface for creating and manipulating tensors. To create tensors you should use the Initialisers. These can be used on their own or chained together with the Setters to create the desired tensor.

For example

Tensor t;
t.ones({3,3}).dType(NTdtypes::kFloat).device(NTdtypes::kGPU);
